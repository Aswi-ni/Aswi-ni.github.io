n8n Automation workflow- Policy Agent

The Policy Agent is an AI-powered chatbot workflow developed using n8n, designed to handle and respond to user queries with contextual understanding and enriched knowledge retrieval. The workflow is triggered when a chat message is received, initiating interaction with an AI Agent powered by the OpenAI Chat Model. To maintain the flow of conversation, a Simple Memory component is used to retain context across interactions.

For broader and more accurate information access, the agent is enhanced with two major tools: Wikipedia, which provides real-time factual data, and a Pinecone Vector Store integrated with OpenAI Embeddings, enabling efficient semantic search via Retrieval-Augmented Generation (RAG). The system utilizes OpenAI's embedding capabilities to convert text into vector representations, allowing for more intelligent and relevant responses.

This project demonstrates the integration of LLMs, vector databases, and workflow automation tools to create an intelligent policy support assistant capable of dynamic, knowledge-rich conversations.
