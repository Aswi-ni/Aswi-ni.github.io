Credit card Sales Voice Agent

The Credit Card Sales Voice Agent is an AI-powered assistant deployed via Vapi and a Twilio number, designed to engage potential customers over phone calls. The assistant guides users through available credit card options, eligibility checks, offers, and assists in initiating the application process—all through a smooth, conversational interface.

The solution is also equipped with a web-based chat interface built using custom HTML, which offers the same intelligent experience in text. A robust Python + FastAPI backend manages conversation flow, application status, and campaign-specific offers. Thanks to the assistant override feature, it can dynamically personalize greetings and card recommendations, creating a trusted, branded experience tailored to each user's financial profile.
